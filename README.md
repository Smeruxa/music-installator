# VK Music Installer

Приложение для macOS и Windows: заходишь во ВКонтакте, смотришь **Мою музыку** и **Недавно прослушанные**, слушаешь превью и скачиваешь выбранные треки в свою папку.

## Что умеет

- Вход через веб-сессию VK (открывается окно логина)
- Два раздела с переключателем сверху
- Выбор треков (в том числе «выбрать все» и drag по чекбоксам)
- Плеер с громкостью
- Скачивание в MP3 (в том числе из HLS через ffmpeg)

## Запуск для разработки

```bash
npm install
npm run dev
```

Полезное рядом:

```bash
npm test
npm run lint
npm run typecheck
```

## Сборка

```bash
npm run dist:mac   # dmg под macOS
npm run dist:win   # установщик Windows (x64)
```

Готовые файлы лежат в `release/`.

Windows-сборку лучше гонять через `npm run dist:win` с Mac/CI — скрипт сам подкладывает `ffmpeg.exe` под Windows. Иначе в инсталлер может попасть mac-бинарник, и скачивание на ПК «успешно» завершится с пустой папкой.

## На заметку

Только для бэкапа **своей** музыки. Пароль нигде не сохраняется — в сейфе Electron остаётся только токен сессии.
